ds_arpes_plugin's documentation
===============================

Welcome! ``ds_arpes_plugin`` is a python module that acts as a plugin to PIT, 
the python image tool provided by the `data_slicer 
<https://github.com/kuadrat/data_slicer>`_ package.
It connects the functionality provided by the `arpys 
<https://github.com/kuadrat/arpys>`_ package to PIT.
